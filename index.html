GitHub Sync — Spreadsheet Database

Provider data is stored as providers.csv in the GitHub repository. Changes made here are saved locally; use Commit to GitHub to push them to the spreadsheet. Anyone with access to the repo can edit the CSV directly in Excel or Google Sheets.

Sync Actions

Workflow: Edit providers on the web tool OR directly in the CSV file in GitHub. After editing the CSV file externally, click Refresh from GitHub to pull changes here. The ✓ Human Verified badge is set via the confirmed column in the spreadsheet or via the provider edit form.

Spreadsheet Format

The CSV uses these columns in order. The doctors field uses | as a separator (e.g. Dr. A|Dr. B). Set confirmed to true to show the verified badge.

id, displayName, practice, address, city, phone, website, specialty, ownership, catComanage, glaucTC, distMain, distEast, doctors, notes, confirmed
