GitHub Token

Required to save any changes (edits, verifications) back to providers.csv on GitHub. Generate a Personal Access Token with repo scope at github.com → Settings → Developer settings → Personal access tokens.

Sync Data

Pull the latest providers.csv from GitHub into this browser session, or push the current in-browser data up to GitHub.

Change Admin Password

Updates the password for the "admin" account. You'll need to log in again afterward.

Monthly Backups

Automatic monthly backups are handled by a GitHub Actions workflow that runs on the 1st of each month and copies providers.csv to the archives/ folder in your repository. The workflow file is .github/workflows/monthly-backup.yml.

To restore from a backup: go to your GitHub repo → archives/ folder → open the dated CSV → copy its content into providers.csv, then use "Push Current Data to GitHub" above (or edit directly on GitHub).
